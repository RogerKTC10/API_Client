# API_Client
Un API CLient de type REST 
Une idée de creation d'un API Personnel REST en java avec utilisation de SPRING

Ce projet est un robuste pour une API REST sécurisée avec Java et Spring Boot. Il sert de fondation pour de futurs projets nécessitant une gestion fine des utilisateurs et des rôles.

Dans cette phase du projet, l'objectif est de créer un noyau sécurisé et autonome. Voici pourquoi la liaison avec des bases publiques n'est pas activée
